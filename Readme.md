# How To Use

`graphtirps.py` uses library `mtnleaderreport.py` to generate graphs of leader activity.  Modify `graphtrips.py` to 
generate the reports that you want.

## Environment

Set up the environment:

	python3 -m venv env
	. env/bin/activate
	pip install -r requirements.txt
